Sie müssen drei leere Kisten mit einer Drohne transportieren, die bei jeder Fahrt nur eine Kiste auf einmal heben kann. Mit anderen Worten: Sie können die drei leeren Kisten immer transportieren, indem Sie drei Fahrten mit der Drohne machen. Aber Sie könnten auch weniger als drei Fahrten machen, wenn Sie eine Kiste in eine andere stecken können. Die Kisten sind würfelförmig und die einzige Einschränkung beim Einsetzen einer Kiste in eine andere ist die Größe, unabhängig vom Gewicht.

Eine Schachtel der Größe X kann in eine Schachtel der Größe Y gestellt werden, wenn X < Y. Beachten Sie also, dass eine Schachtel nicht in eine andere der gleichen Größe passt. Außerdem können zwei Schachteln der Größen X und Y nebeneinander in eine Schachtel der Größe Z gestellt werden, wenn (X+Y) < Z.
